# bookmarker-js
Bookmarker Tool with JS and HTML5 Local Storage

## Current Implementations
1. Addition and Deletion of Bookmarks
2. Cool Design
3. Responsive for all devices
4. Permanent bookmarks data in HTML5 Local Storage
5. Search Options
6. Edit Bookmarks
